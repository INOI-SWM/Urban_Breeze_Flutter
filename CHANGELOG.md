# Changelog

## [1.0.0] (2025-10-10)

### ✨ 주요 기능

#### 🔗 통합 연동

- **Garmin Connect** 연동
- **Suunto** 연동
- **Apple Health Kit** 데이터 동기화
- Terra API 기반 헬스 데이터 통합

#### 🏃 운동 기록

- 운동 기록 조회 및 상세 정보
- 고도 그래프, 심박수, 속도 등 상세 지표
- GPX 파일 다운로드
- 운동 기록 제목 수정 및 삭제

#### 🗺️ 경로 추천

- 추천 경로 목록 및 상세 정보
- 거리/경사도 필터링
- 딥링크 기반 경로 공유

#### 👤 계정 관리

- 소셜 로그인 (카카오, 구글, 애플)
- 프로필 이미지 업로드/수정
- 회원 탈퇴

#### 📊 통계

- 라이딩 통계 대시보드
- 총 거리, 시간, 칼로리 집계
